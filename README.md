# ✨ Features

- 🎥 Fullscreen cinematic hero banners
- 🔥 Trending, Top Rated & Popular movies
- 🎬 Auto-playing movie trailers
- 🧠 AI movie recommendation assistant
- ❤️ Favorites system
- 📌 Watchlist system
- 🎞 Hover trailer previews
- 🔐 Authentication system
- 🌙 Netflix-inspired dark UI
- 📱 Fully responsive design
- ⚡ Dynamic TMDB API integration
